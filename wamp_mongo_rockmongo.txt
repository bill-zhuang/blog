windows下 php apache mysql mongo rockmongo 安装

php 安装
1. 在http://windows.php.net/download/上下载(非exe安装，如要exe可执行文件安装，则为http://www.php.net/downloads.php)
根据需要现在线程安全或者非安全版，
2. 下载后解压到所需目录下，将目录下的php.ini-development重命名为php.ini
3. 配置php.ini
找到extension_dir这一行，设置php目录下ext文件所在路径
如extension_dir = "D:/WAMP/php-5.5.6-Win32-VC11-x64/ext"
找到extension=xxx.dll，在需要的功能钱去掉';'
设置date.timezone = Asia/Shanghai

apache的安装
1.在http://windows.php.net/download/上找到的apache下载链接(网站上不推荐在apache.org上下载)：http://www.apachelounge.com/download/
根据自己需要下载32/64位版本，下载的apache已经说明了用vc9还是11编译的，跟下一步相关
2.如果电脑上没安装vc9/11所需编译文件，根据上一步下载的vc9/11编译的apache选择下载相应的文件
下载链接为(微软官方)：http://www.microsoft.com/en-us/download/details.aspx?id=30679
3.解压下载的文件后可以看到有ReadMe.txt文件，按说明一步一步来，
解压到目录后，配置目录下conf文件夹中的httpd.conf文件，如果解压到的c:/Apache24下，则大部分不需修改，否则需修改下列部分：
用ctrl+F 搜索(已本人为例)

ServerRoot
加上Listen 80
LoadModule php5_module "D:/WAMP/php-5.5.6-Win32-VC11-x64/php5apache2_4.dll"
PHPIniDir "D:/WAMP/php-5.5.6-Win32-VC11-x64"
DocumentRoot "D:/WAMP/Apache24/htdocs"
ScriptAlias /cgi-bin/ "D:/WAMP/Apache24/cgi-bin//"
<IfModule dir_module>
    DirectoryIndex index.php index.htm index.html
</IfModule>
<Directory "D:/WAMP/Apache24/cgi-bin">
    AllowOverride None
    Options None
    Require all granted
</Directory>

修改完后在dos命令行下进入解压目录下的bin目录下
httpd.exe -k install //注册apache服务
然后将bin目录下的ApacheMonitor.exe创建快捷方式到桌面或者快速启动栏等

mongodb的安装
1.在 http://www.mongodb.org/downloads 下选择自己os下的下载链接，这里是window，因为os是32位，选择32的，
有zip和msi两种，这里选择的是zip，看了下msi的较大，100多m，
2.下载zip文件后解压，在命令行模式下(cmd)下进入解压后文件下的bin目录，输入以下命令
mongod --dbpath D:\MongoDB\db //这里加了dbpath的原因是，默认不加的话，数据库是安装在C:\data\下，
建议还是设置自己数据库存放位置，上述步骤后就ok了
3. 由于每次启动mongodb都要运行上一步的命令，比较麻烦，注册为windows系统服务，进入命令行模式，cmd
e:
cd software\mongodb-win32-i386-2.4.10\bin 
E:\software\mongodb-win32-i386-2.4.10\bin>mongod.exe --dbpath "D:\MongoDB\db" 
--logpath "D:\MongoDB\log\mongolog.txt" --logappend –install

rockmongo的安装
1. 在 http://rockmongo.com/downloads 下下载 RockMongo v1.1.5，RockMongo-on-Windows v0.0.4没试过
2. 下载后解压到所要目录下
3. 为了让php 支持mongdb，需下载mongdb.dll，
http://www.php.net/manual/en/mongo.installation.php php官网提供所有os所需的mongdb扩展所需的文件，
这里选择windows的，下载url：https://s3.amazonaws.com/drivers.mongodb.org/php/index.html
选一个下载即可，看自己需要，下载后解压重命名为php_mongo.dll，复制到php的ext目录下，
在php.ini文件中加上extension=php_mongo.dll这句即可
4. 配置apache
修改/etc/httpd/conf/httpd.conf文件，添加如下内容（我自己的配置，看自己需要修改）：
<VirtualHost *:80>
ServerAdmin  mongo.com
DocumentRoot D:/rockmongo-1.1.5/rockmongo
ServerName mongo.com

ErrorLog "|D:/Wamp/Apache24/bin/rotatelogs.exe D:/Wamp/Apache24/logs/mongo.com-error_%Y_%m_%d.log 86400.log"
CustomLog "D:/Wamp/Apache24/logs/mongo.com-access.log" common

<Directory />
Options Indexes FollowSymLinks
AllowOverride  All
Order allow,deny
Allow from all
</Directory>
</VirtualHost>
5. 重启apache，浏览器中输入mongo.com(上面我设置的ServerAdmin及ServerName的值)，用户名和密码都为admin，就可以使用了
ps：
1.如果出现错误提示： ini_set(): To prevent data corruption, you are not allowed to turn on the mongo.native_long setting on 32-bit platforms in
只需在下载的rockmongo目录下的index.php文件，注释掉
ini_set("mongo.native_long", 1);即可
2. 如果登录后页面上显示错误，但是rockmongo还是能用，在下载的rockmongo目录下的config.php文件加上
ini_set("display_errors",0);即可
3. rockmong的详细配置参考官方url：http://rockmongo.com/wiki/configuration?lang=zh_cn
