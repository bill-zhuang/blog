1. using 与 use
using 用在join操作中，如果两个表join的条件相同(字段名相同)，可以使用using
	如：...from table_a join table_b on table_a.id = table_b.id...  等价于 ...from table_a join table_b using (id)...
use 用在sql查询中告诉mysql使用那个索引
	如：...from table use index(index_name)...
